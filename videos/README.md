# 马年串词视频说明

## 视频文件

请将马年串词视频文件放置在 `videos/` 目录下：

**文件名**: `horse-year-intro.mp4`

## 视频规格建议

- **格式**: MP4 (H.264 编码)
- **分辨率**: 
  - 移动端: 720p (1280x720) 或 1080p (1920x1080)
  - 建议：16:9 或 9:16 (竖屏更适合H5)
- **时长**: 建议 30秒 - 2分钟
- **码率**: 建议 2-5 Mbps (平衡质量和加载速度)
- **文件大小**: 建议 20MB 以内（适合移动端加载）
- **帧率**: 30fps

## 视频内容建议

### 开场部分
- 马年主题动画或实拍画面
- 欢快的背景音乐
- "2026马年快乐" 等标题文字

### 主持串词部分
- 中文主持人出镜或画外音
- 马年祝福话语（如"马到成功"、"龙马精神"等）
- 过渡自然，引导用户点击生成卡片

### 结尾部分
- "点击下方按钮生成你的专属祝福卡片" 提示
- 画面淡出或定格在提示画面

## 兼容性要求

视频需要支持以下特性：
- ✅ iOS Safari 播放
- ✅ Android Chrome 播放
- ✅ 微信内置浏览器播放
- ✅ 支持点击播放
- ✅ 支持播放结束事件

## 优化建议

1. **压缩优化**: 使用工具（如 HandBrake）压缩视频，减小文件体积
2. **编码格式**: 使用 H.264 编码，兼容性最好
3. **音频格式**: 使用 AAC 编码
4. **移动端优化**: 考虑使用竖屏（9:16）更符合H5浏览习惯
5. **预加载**: 系统会在页面加载时预加载视频

## 临时测试

在正式视频准备好之前，你可以：
1. 使用任何MP4视频文件重命名为 `horse-year-intro.mp4`
2. 测试视频播放和卡片生成流程
3. 后续替换为正式的马年串词视频

## 推荐工具

- **视频压缩**: HandBrake (免费开源)
- **视频剪辑**: 剪映、Premiere Pro
- **格式转换**: FFmpeg、在线转换工具

## 注意事项

- 视频文件名必须严格为 `horse-year-intro.mp4`
- 确保视频有清晰的结束点，以便触发生成按钮
- 背景音乐不要太吵，不影响主持人的声音
- 画面明亮，适合各种光线环境下观看
