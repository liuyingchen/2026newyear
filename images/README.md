# 马年祝福卡片图片说明

## 需要准备的5张卡片图片

请将以下5张图片放置在 `images/` 目录下：

1. **card-1.jpg** - 马到成功主题
2. **card-2.jpg** - 龙马精神主题
3. **card-3.jpg** - 金马报喜主题
4. **card-4.jpg** - 马年大吉主题
5. **card-5.jpg** - 福马迎春主题

## 图片规格建议

- **尺寸**: 800x1067 像素（3:4 比例）
- **格式**: JPG 或 PNG
- **文件大小**: 每张建议不超过 500KB
- **分辨率**: 建议 72dpi（适合H5展示）
- **内容**:
  - 马年主题元素（马、梅花、中国结等）
  - 红色、金色等喜庆色调
  - 简洁大方的设计风格

## 临时占位图

在正式图片准备好之前，系统会使用SVG占位图。
占位图内容为卡片标题文字。

## 使用说明

1. 将你准备的5张卡片图片重命名为 `card-1.jpg` 到 `card-5.jpg`
2. 放置在本目录（`images/`）下
3. 刷新页面即可看到真实卡片

## 设计建议

- 采用中国传统风格，融入马年元素
- 使用喜庆的红色和金色为主色调
- 背景简洁，突出祝福主题
- 可以添加装饰元素如：灯笼、鞭炮、梅花等
- 确保文字清晰可读（如果图片上需要文字）
